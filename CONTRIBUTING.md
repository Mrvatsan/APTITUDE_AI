# Contributing to Aptitude Master

First off, thank you for considering contributing to Aptitude Master! It's people like you that make Aptitude Master such a great tool.

## Code of Conduct
By participating in this project, you are expected to uphold our Code of Conduct.

## How Can I Contribute?
- **Reporting Bugs**: Use GitHub Issues.
- **Suggesting Enhancements**: Use GitHub Issues.
- **Pull Requests**: Pull requests are always welcome!

## Style Guide
- Use consistent indentation (4 spaces).
- Document all new functions with JSDoc headers.
- Include descriptive logging for new logic paths.
